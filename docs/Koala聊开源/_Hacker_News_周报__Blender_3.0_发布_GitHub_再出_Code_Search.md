Apache Guacamole 是一个无客户端的远程桌面网关，像常见的 VNC，RDP 和 SSH 等协议，它都可以支持。之所以被称为无客户端，是因为它既不需要安装插件，也不需要安装桌面客户端软件，直接使用 web 打开，它就可以工作。如果你过去是 VNC，RDP 和 SSH 远程桌面的使用者，那你可以考虑尝试一下这个软件。
